# GitHub Upload Bundle

This folder is a cleaned bundle for uploading the EMIT simulator to GitHub for reproducible testing.

## Quick start
1. Create a new GitHub repo.
   Repo URL: https://github.com/neal2hawkins-hub/EMIT_SIM
2. Upload all contents of this folder.
3. Install deps:
   pip install -r requirements.txt
4. Run:
   python main.py --size 256

## Analysis
python analysis/analyze_tests.py --runs_dir runs
